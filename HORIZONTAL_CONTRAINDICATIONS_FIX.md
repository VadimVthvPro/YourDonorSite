## 📊 ЧТО ИЗМЕНИЛОСЬ В CSS:

### 1. Карточка - `flex-direction: row`
```css
.contra-card {
    display: flex;
    flex-direction: row;  /* ВСЁ ГОРИЗОНТАЛЬНО */
    align-items: center;
    gap: 12px;
}
```

### 2. Хедер - компактный
```css
.contra-card-header {
    width: auto;        /* Не на всю ширину */
    flex-shrink: 0;     /* Не сжимается */
}
```

### 3. Список - `flex-direction: row`
```css
.contra-list {
    display: flex;
    flex-direction: row;   /* ЭЛЕМЕНТЫ В РЯД */
    flex-wrap: wrap;       /* Перенос если не влезает */
    gap: 6px 10px;
}

.contra-list li {
    display: inline-block; /* В СТРОКУ */
    white-space: nowrap;   /* Не переносим текст */
}
```
