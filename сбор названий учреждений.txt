Отлично 👍
Тогда самый надёжный путь — автоматически собрать все больницы и поликлиники из OpenStreetMap (там как раз лежат все метки на карте), а потом разложить их в JSON формата:
область
 └ район
    ├ объект 1
    ├ объект 2
    └ объект 3


Ниже — готовый рабочий скрипт. Ты запускаешь его у себя, и он сам выгружает все медучреждения Беларуси и сохраняет итоговый JSON-файл.
________________


✅ Что делает скрипт
Он:
✔ берёт границу Беларуси из OpenStreetMap
✔ находит все объекты с тегами:
* amenity = hospital
* amenity = clinic
* amenity = doctors
* healthcare = hospital / clinic / polyclinic
✔ для каждой точки определяет:
* название
* район
* область
✔ сортирует их в структуру:
{
  "Минская область": {
    "Минский район": [
      { "name": "...", "type": "hospital" }
    ]
  }
}


✔ сохраняет файл belarus_medical.json
Контакты, телефоны и адреса не берутся — только то, что ты просил.
________________

Если Python ещё не установлен — ставь с python.org.
Потом в терминале:
pip install osmnx geopandas shapely reverse_geocoder tqdm

🧩 Готовый код
Скопируй целиком в файл, например:
build_medical_json.py
import osmnx as ox
import geopandas as gpd
import reverse_geocoder as rg
from collections import defaultdict
from tqdm import tqdm
import json


print("Загружаем границы Беларуси...")
country = ox.geocode_to_gdf("Belarus")


TAGS = {
    "amenity": ["hospital", "clinic", "doctors"],
    "healthcare": ["hospital", "clinic", "polyclinic"]
}


print("Загружаем объекты медицины из OpenStreetMap...")
gdf = ox.features_from_polygon(country.geometry.iloc[0], TAGS)


gdf = gdf[gdf.geometry.notnull()].copy()
gdf["lat"] = gdf.geometry.centroid.y
gdf["lon"] = gdf.geometry.centroid.x


coords = list(zip(gdf["lat"], gdf["lon"]))
print("Определяем регионы и районы...")
geo_results = rg.search(coords)


result = defaultdict(lambda: defaultdict(list))


for idx, row in tqdm(gdf.iterrows(), total=len(gdf)):
    geo = geo_results[list(gdf.index).index(idx)]
    
    region = geo.get("admin1", "Неизвестная область")
    district = geo.get("admin2", "Неизвестный район")
    
    name = row.get("name")
    if not name:
        continue


    obj_type = (
        row.get("healthcare")
        or row.get("amenity")
        or "medical"
    )


    entry = {
        "name": name,
        "type": obj_type
    }


    result[region][district].append(entry)


final_data = {region: dict(districts) for region, districts in result.items()}


output_file = "belarus_medical.json"
with open(output_file, "w", encoding="utf-8") as f:
    json.dump(final_data, f, ensure_ascii=False, indent=2)


print(f"\n✅ Готово! Файл сохранён как: {output_file}")
print("Количество областей:", len(final_data))

▶️ Как запустить
В папке с файлом выполни:
python build_medical_json.py


Через пару минут появится файл:
belarus_medical.json


В нём будет вся страна, отсортированная по областям и районам.
